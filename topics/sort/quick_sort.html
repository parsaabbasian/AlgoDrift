How it Works

Concept: Quick Sort is a divide-and-conquer algorithm. It picks an element as a pivot and partitions the given array around the picked pivot.

Steps:

  • Choose Pivot: In this visualization, we select the last element as the pivot.
  • Partition: Reorder the array so that all elements smaller than the pivot come before it, and all elements greater come after it.
  • Recursion: Recursively apply the above steps to the sub-array of elements with smaller values and separate sub-array of elements with greater values.
