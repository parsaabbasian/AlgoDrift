function insertionSort(arr) {  for (let i = 1; i < arr.length; i++) {    let key = arr[i];    let j = i - 1;    // Move elements > key    while (j >= 0 && arr[j] > key) {      arr[j + 1] = arr[j];      j--;    }    arr[j + 1] = key;  }}
How it Works

Concept: Insertion Sort builds the sorted array one item at a time. It is much like sorting playing cards in your hand.

Steps:

  • Pick Key: It starts from the second element (the key) and compares it with the elements before it.
  • Shift: If the key is smaller than the previous element, it shifts the previous element to the right.
  • Insert: This repeats until the correct position for the key is found, and it is inserted there.
