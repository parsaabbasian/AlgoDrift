function selectionSort(arr) {  const n = arr.length;  for (let i = 0; i < n; i++) {    let min = i;    for (let j = i + 1; j < n; j++) {      // Find minimum      if (arr[j] < arr[min]) {        min = j;      }    }    if (min !== i) swap(i, min);  }}
How it Works

Concept: Selection Sort is a simple comparison-based algorithm. It divides the array into a sorted part and an unsorted part.

Steps:

  • Find Minimum: It scans the unsorted part to find the smallest element.
  • Swap: The smallest element is swapped with the first element of the unsorted part.
  • Repeat: The sorted boundary moves one step to the right, and the process repeats until the array is fully sorted.
